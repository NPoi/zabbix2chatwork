ZabbixからChatWorkへ投稿するためのスクリプト

Usage
-------

```
$ zabbix2chatwork {{ API_KEY }}:{{ CHATWORKS_ROOM_NAME }}:{{ NOTIFICATION_USER_ID1 }}[,{{ NOTIFICATION_USER_ID2 }},...] \
{{ MESSAGE_TITLE }} \
{{ MESSAGE_BODY }}
```
